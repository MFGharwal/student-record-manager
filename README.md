# Student Record Manager (C++)

A simple console-based student management program built using C++.  
This project demonstrates object-oriented programming, multi-file structure, vectors, and basic CLI interaction.

## Features
- Add a new student (name, ID, GPA)
- List all students
- Search for a student by ID

## How to Compile & Run



```
g++ main.cpp student.cpp -o student_manager
./student_manager
```



---


## Technologies
- C++
- STL vectors
- Object-Oriented Programming (classes, encapsulation)

## Notes
This project was built to practice C++ fundamentals such as multi-file project structure, classes, and user input handling.
