\# Student Record Manager (C++)



A simple console-based student management program built in C++.  

This project demonstrates object-oriented programming, multi-file C++ structure, vectors, and basic CLI interactions.



---



\## Features

\- Add new students (name, ID, GPA)

\- List all students

\- Search for a student by ID



---



\## How to Compile \& Run



```
g++ main.cpp student.cpp -o student_manager
./student_manager
```



---



\## Technologies

\- C++

\- STL vectors

\- Object-Oriented Programming (classes, encapsulation)



---



\## Notes

This project was built to practice C++ fundamentals such as OOP, header/source file separation, vectors, input handling, and menu-driven CLI design.
